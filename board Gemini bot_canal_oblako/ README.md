⚓ Дошка оголошень Одеса (Инструкция)
Этот проект состоит из Бот-админа, Web App (сайта) и Канала-ленты.
🛠 Настройка Переменных Окружения (.env)
При деплое на Render.com обязательно добавьте эти переменные:
 * MONGO_URI — ссылка на вашу базу данных MongoDB Atlas.
 * BOT_TOKEN — токен от @BotFather.
 * ADMIN_ID — ваш цифровой ID (узнать в @userinfobot).
 * CHANNEL_ID — ID вашего канала (начинается с -100...).
📱 Команды управления в Telegram
Эти команды вводите прямо в чат с вашим ботом (доступны только вам как админу).
💳 Работа с картами (Ротатор)
 * /add_card [номер] [название] — добавить новую карту в ротацию.
   * Пример: /add_card 4441000011112222 Монобанк
 * /add_phone [номер] [оператор] — добавить номер телефона для пополнения.
   * Пример: /add_phone 0631234567 Лайф
 * /wallets — посмотреть список всех активных реквизитов и сколько раз они были выданы клиентам.
 * /reset_limits — обнулить счетчики выдач (рекомендуется делать 1-го числа каждого месяца).
📢 Управление контентом
 * Пока что добавление рекламы (слайдер сверху) и удаление объявлений происходит через прямую правку в базе MongoDB или через кнопки «🗑 ВИДАЛИТИ» в уведомлениях админа.
🔄 Жизненный цикл объявления
 * Подача: Клиент заполняет форму в Web App.
 * Оплата: Сайт берет из базы карту, которую выдавали меньше всего, и показывает её клиенту.
 * Проверка: Бот присылает вам сообщение: "Нове замовлення: ID-1234".
 * Активация: Вы проверяете поступление денег на карту/телефон. Если ок — жмете кнопку «✅ АКТИВУВАТИ».
 * Финал: Объявление становится видимым на сайте и автоматически пересылается в ваш Telegram-канал.
⚠️ Важные технические нюансы
 * SSL: Telegram Mini Apps работают только через https. Убедитесь, что ваш домен на Render имеет активный сертификат (Render делает это бесплатно).
 * CORS: Если сайт не грузит данные, проверьте, что в server.js разрешены запросы с вашего домена.
 * Хранение фото: В текущей версии фото для объявлений не предусмотрены (только текст). Если захотите добавить фото, нужно будет подключить Cloudinary или Firebase Storage.
💡 Советы по налогам и безопасности
 * IBAN: Если сумма оплаты больше 1000 грн, лучше выдайте клиенту IBAN вручную через чат.
 * Лимиты: Не держите в ротации более 30-50 оплат на одну карту в месяц. Как только лимит близок — выключайте карту командой (или она сама уйдет в конец очереди благодаря sort({ useCount: 1 })).
